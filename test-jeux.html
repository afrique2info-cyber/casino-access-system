Test via le Système Complet

Comment tester le flux complet :

1. Configurez XAMPP/WAMP - Placez le dossier dans htdocs/

2. Importez la base de données - Exécutez database.sql dans phpMyAdmin

3. Générez un code admin - Accédez à /admin/ (admin/admin123)

4. Testez comme joueur - Entrez le code sur la page d'accueil

5. Jouez aux jeux - Sélectionnez KENO ou SLOTS depuis le dashboard
